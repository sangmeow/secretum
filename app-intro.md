## FAQ (자주 묻는 질문)

### Q1. 데이터가 외부로 전송되나요?
```
아니요. Secretum은 네트워크 통신 코드가 원천적으로 존재하지 않습니다.
모든 데이터는 당신의 기기 내부에만 저장되며, 외부로 나갈 수 없습니다.
```

### Q2. PIN을 잊어버리면 어떻게 하나요?
```
안타깝게도 PIN을 잊어버리면 데이터 복구가 불가능합니다.
이는 보안을 위한 설계이므로 PIN을 안전하게 기억해 주세요.
```

### Q3. 여러 기기에서 사용할 수 있나요?
```
Secretum은 완전 오프라인 앱으로 클라우드 동기화를 제공하지 않습니다.
각 기기마다 독립적으로 사용해야 합니다.
```

### Q4. 기기를 바꾸면 데이터를 옮길 수 있나요?
```
기기 전체 백업(Android Google 백업, iOS iCloud 백업)을 통해
앱 데이터를 함께 백업하고 복원할 수 있습니다.
```

### Q5. 왜 네트워크 권한을 요청하지 않나요?
```
Secretum은 인터넷 연결이 필요 없는 완전 오프라인 앱입니다.
네트워크 코드 자체가 없으므로 네트워크 권한이 불필요합니다.
```

### Q6. 무료인가요? 광고가 있나요?
```
완전 무료이며 광고가 전혀 없습니다.
숨겨진 비용이나 인앱 구매도 없습니다.
```
